# 📊 Vizora AI
From Data to Decisions, Instantly.

Vizora AI is a Streamlit-based analytics engine that transforms raw CSV/XLSX data into clean insights, dashboards, and exportable reports.

## 🚀 Features

- 📂 Upload CSV / Excel datasets
- 🧹 Automatic data cleaning & validation
- ⚙️ Data transformation engine
- 📊 Auto dashboard generation
- 📤 Export as CSV / JSON / PDF
- 🧠 Smart data understanding pipeline
- 
## 🧠 Architecture

Upload → Cleaning → Transformation → Dashboard → Export

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- SQLite
- ReportLab
